How to clone this repo
----
```sh
git clone https://github.com/ciawalsh/instagram-challenge.git
```

How to run tests
----
```sh
cd instagram-challenge
bundle install
rspec
```

How to run locally
----

```sh
rails server
```

Visit http://localhost:3000/
